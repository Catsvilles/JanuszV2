# Janusz V2

Multifunctional chat bot.

![Architecture Diagram](/github-data/Diagram2.png?raw=true)

## Features

- Discord support
- Mumble support
- Interactive web panel
- Node-based GUI audio system
- Text to speach
- Sound board support
- Basic YouTube support
- Modular architecture
- Hot-Reload everything!

## Usage

### Run development

```bash
npm run start
```

### Build production

```bash
npm run build:prod
```

### Start production

```bash
npm run start:prod
```
